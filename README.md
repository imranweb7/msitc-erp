System Requrements
============================
Your server need to meet the following requirements in order to run Freelance Cockpit 2.

PHP 5.3+
MYSQL 5+
PHP Mcrypt Extention
PHP MBString Extension
PHP DOM Extension
PHP GD Extension
PHP MYSQL Extension
PHP PDO Extention
PHP CURL Extention
Cronjob (in order to activate recurring invoices)
